EventUtils

EventUtils is a powerful Minecraft plugin designed for Minecraft servers to display event alerts, modloader/client detection, and more.

  • Display custom event alerts in chat and action bar
  • Real-time Minecraft client detection (Lunar, Feather, Vanilla, MultiMC)
  • Highly customizable alert messages
  • Easy integration with other server plugins
  • Optimized for performance with low resource usage
